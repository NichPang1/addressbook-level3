PROJECT: AddressBook - Level 3

Overview

MediBook is adapted from Software Engineering Education’s (SE-EDU) AddressBook Level 3 application and it has been designed specifically for healthcare professionals. It contains approximately three thousand lines of code (LoC) and it is written in an object-oriented programming (OOP) fashion of the Java programming language.

Summary of contributions

Together with my project group mates, Xi Yi, Nicholas and Jia Jian, MediBook was born. The following is a summary of my contributions.

  • Major enhancement: MediChat - allows instant messaging between healthcare professionals

    • What it does:

      • Allows users to send messages instantly in a chatroom for multiple users to view

      • View previously sent messages after logging back in (coming soon)

      • Opens in a separate window so that users can continue working in MediBook without the need to close MediChat

    • Justification:

      • Healthcare professionals are often overloaded and thus do not have time to spend on email threads which are confusing.

      • Colleagues might also get accidentally left out receiving the carbon copy and thus cause a communication breakdown.

      • Although there are instant messaging channels already available such as WhatsApp and Telegram, MediChat is meant to be used within the facility’s network so that there is minimal risk of a data breach.

    • Highlights:

      • As instant messaging channels are readily available already, there is little documentation available regarding the implementation of this feature.
